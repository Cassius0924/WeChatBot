# 基于WxBotWebhook的微信机器人

## 介绍

基于[wechatbot-webhook](https://github.com/danni-cool/wechatbot-webhook)

## 设置管理员

在`config.py`中填写管理员名称。

管理员接收机器人状态变化通知。

## 功能

- [x] GPT问答，基于[gpt4free](https://github.com/xtekky/gpt4free)
- [x] 获取B站热搜
- [x] 获取知乎热搜
- [x] 获取微博热搜
- [x] 获取抖音热搜
- [x] 获取GitHub趋势
- [x] 单词/词语翻译
